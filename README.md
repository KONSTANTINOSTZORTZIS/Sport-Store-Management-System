
# 🛒 Σύστημα Διαχείρισης Αθλητικού Καταστήματος 
Sport Store Management System | A standalone desktop application built with Python and Tkinter, featuring a local SQLite database for inventory and customer relationship management.

## ✨ Λειτουργίες
* **Διαχείριση Δεδομένων**: Εύκολη εισαγωγή και προβολή για Πελάτες, Προϊόντα και Παραγγελίες.
* **Έτοιμα SQL Queries**: Ενσωματωμένη εκτέλεση και των 7 σύνθετων ερωτημάτων της εργασίας (π.χ. πωλήσεις ανά πωλητή, αποθέματα, μέσοι όροι).
* **Φιλικό GUI**: Περιβάλλον με καρτέλες (Tabs) και drop-down menus που διασφαλίζουν την ορθότητα των δεδομένων.

## 🛠️ Τεχνικά Χαρακτηριστικά
* **Γλώσσα**: Python 3.10+.
* **Βάση Δεδομένων**: SQLite3 (δεν απαιτείται εγκατάσταση server).
* **GUI**: Tkinter (περιλαμβάνεται στην Python).

## 🚀 Οδηγίες Εγκατάστασης για Χρήστες
Για να τρέξετε την εφαρμογή στον υπολογιστή σας, ακολουθήστε τα εξής βήματα:

1. **Εγκατάσταση Python**: Κατεβάστε την Python 3 από το [python.org](https://www.python.org/). Κατά την εγκατάσταση, επιλέξτε το **"Add Python to PATH"**.
2. **Λήψη Αρχείων**: Κατεβάστε όλα τα αρχεία του project σε έναν φάκελο στην επιφάνεια εργασίας σας.
3. **Εκτέλεση**: Ανοίξτε το τερματικό μέσα στον φάκελο και γράψτε:
   python app.py
