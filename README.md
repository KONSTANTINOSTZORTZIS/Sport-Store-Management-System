
# 🛒 Σύστημα Διαχείρισης Αθλητικού Καταστήματος 
Sport Store Management System | A standalone desktop application built with Python and Tkinter, featuring a local SQLite database for inventory and customer relationship management.

## ✨ Λειτουργίες
* **Διαχείριση Δεδομένων**: Εύκολη εισαγωγή και προβολή για Πελάτες, Προϊόντα και Παραγγελίες.
* **Έτοιμα SQL Queries**: Εκτέλεση 7 προκαθορισμένων ερωτημάτων για την άντληση συγκεκριμένων πληροφοριών από τη βάση.
* **Φιλικό GUI**: Περιβάλλον με καρτέλες (Tabs) και drop-down menus που διασφαλίζουν την ορθότητα των δεδομένων.

## 🛠️ Τεχνικά Χαρακτηριστικά
* **Γλώσσα**: Python 3.10+.
* **Βάση Δεδομένων**: SQLite3 (δεν απαιτείται εγκατάσταση server).
* **GUI**: Tkinter (περιλαμβάνεται στην Python).

## 🚀 Οδηγίες Εγκατάστασης & Εκτέλεσης
Για να λειτουργήσει η εφαρμογή στον υπολογιστή σας, ακολουθήστε τα παρακάτω βήματα με τη σειρά:

1. **Λήψη Αρχείων**
Κατεβάστε και αποσυμπιέστε το project. Βεβαιωθείτε ότι ο φάκελος περιλαμβάνει τα εξής απαραίτητα αρχεία:


**app.py**: Η κύρια εφαρμογή.

**database_setup.py**: Το script αρχικοποίησης της βάσης.

**create_new.sql & insert_new.sql**: Τα σενάρια δημιουργίας πινάκων και δεδομένων.

2. **Εγκατάσταση Python**
Βεβαιωθείτε ότι έχετε εγκατεστημένη την Python 3.10 ή μεταγενέστερη από το [python.org](https://www.python.org/) .

Σημαντικό: Κατά την εγκατάσταση, επιλέξτε το πλαίσιο "Add Python to PATH".

3. **Αρχικοποίηση Βάσης Δεδομένων** (Πρώτη Φορά)
   
   i. Πλοήγηση στον κατάλογο Source
   Αφού αποσυμπιέσετε το αρχείο ZIP, εισέλθετε στον φάκελο που περιέχει τον κώδικα **(source)** και με δεξί κλικ ανοίξτε το τερματικό σας (PowerShell ή Command Prompt) 

   Σημείωση: Αν χρησιμοποιείτε VS Code, επιλέξτε File > Open Folder και ανοίξτε απευθείας τον φάκελο source.

   ii. Προετοιμασία Βάσης Δεδομένων
   Πριν την πρώτη εκτέλεση της εφαρμογής, πρέπει να δημιουργηθεί η βάση δεδομένων και να φορτωθούν τα αρχικά δεδομένα. Εκτελέστε την εντολή:

   Bash
   python database_setup.py

4. **Εκτέλεση της Εφαρμογής**
Αφού ολοκληρωθεί το setup, μπορείτε να ξεκινήσετε το γραφικό περιβάλλον (GUI) τρέχοντας:

Bash
python app.py
